Spotify Dates

  • encodeSpotifyDate(_:datePrecision:forKey:)

    Encodes a Sate to a date-string in one of the following formats, depending on the datePrecision.

    The expected values for datePrecision are:

    • “YYYY-MM-DD” if datePrecision == “day”
    • “YYYY-MM” if datePrecision == “month”

    • “YYYY” if datePrecision == “year” or == nil.

    Throws

    If the date string could not be encoded into the container for the given key.
