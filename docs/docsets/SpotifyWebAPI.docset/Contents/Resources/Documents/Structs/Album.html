Album

public struct Album : Hashable
extension Album: Codable

A Spotify album.

  • name

    The name of the album.

    In case of an album takedown, the value may be an empty string.

    Declaration

    Swift

    public let name: String
  • tracks

    The tracks of the album.

    For certain endpoints, this property may be nil, especially if it is nested inside a much larger object. For example, it will be nil if retrieved from the search endpoint or if nested inside a Track. When this property is nil, use albumTracks(_:market:limit:offset:) instead, passing in the URI of this album.

    Declaration

    This field is present when getting an artist’s albums.
