    The country of the user, as set in the user’s account profile. An ISO 3166-1 alpha-2 country code.

    This field is only available for the current user and requires the userReadPrivate scope.

    The user’s email address, as entered by the user when creating their account.

    This field is only available for the current user and requires the userReadEmail scope.
