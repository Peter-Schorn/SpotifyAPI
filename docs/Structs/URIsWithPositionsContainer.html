URIsWithPositionsContainer

public struct URIsWithPositionsContainer : Codable, Hashable

A container that holds URIWithPositions and, optionally, the snapshot id of the playlist that the items associated with the URIs (usually tracks/episodes) are contained in.

Used in the body of SpotifyAPI.removeSpecificOccurencesFromPlaylist(_:of:).

This is used for removing specific occurences of items from a playlist. The positions of the URIs are necessary in case the playlist has duplicate items.

Compare with URIsContainer.

Read more at the Spotify web API reference.
