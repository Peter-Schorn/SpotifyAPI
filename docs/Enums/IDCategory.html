IDCategory

public enum IDCategory : String, CaseIterable, Codable, Hashable

A Spotify ID Category, which is the identifier that appears near the beginning of a Spotify URI.

In this URI:

"spotify:track:6rqhFgbbKwnb9MLmUQDhG6"

“track” is the id category.

The id categories are:

  • artist
  • album
  • track
  • playlist
  • show
  • episode
  • local
  • user
  • genre
  • unknown

Read more at the Spotify web API reference.
