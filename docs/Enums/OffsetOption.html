OffsetOption

public enum OffsetOption
extension OffsetOption: Codable
extension OffsetOption: Hashable

Indicates where in the context playback should start. See PlaybackRequest.

One of the following:

  • position(Int): The index of the item in the context at which to start playback. Cannot be used if the context is an artist.
  • uri(SpotifyURIConvertible): The URI of the item in the context to start playback at.
